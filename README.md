## Face, Eye & Smile Detection | OpenCV Project
Real-time detection using Haarcascade Classifiers and WebCam.

### Features
- Detects Face in real-time
- Identifies Eyes and Smile
- Displays labels on screen

### Tools & Libraries
Python, OpenCV, Haarcascade

### How to Run
1. Install dependencies: pip install opencv-python
2. Run: python Face_eye_smile.py
3. Press 'q' to exit
